# 輪播逐頁文 Markdown 格式

Use this structure in Stage 1. Keep visible copy exact and concise; keep production notes separate from visible text.

```md
# <主題>｜輪播逐頁文

狀態：待確認
頁數：<N>

## 第 1 頁｜封面

- 目的：用一句具體痛點吸引老師停留
- 底圖：`cover.png`
- 截圖：無／`assets/<filename>`
- 畫面文字：
  - 標題：「」
  - 副標：「」

## 第 2 頁｜<步驟名稱>

- 目的：
- 底圖：`body.png`
- 截圖：`assets/<filename>`
- 畫面文字：
  - 標題：「」
  - 說明：「」
  - 提醒：「」

## 收尾頁｜<行動呼籲>

- 底圖：`final.png`
- 連結：<URL；若要 QR code 必須明確列出>
- 畫面文字：
  - 標題：「」
  - 說明：「」
```

Before asking for confirmation, check that every screenshot has a page assignment, every visible string is Traditional Chinese, and the proposed sequence can be understood without the source conversation.
